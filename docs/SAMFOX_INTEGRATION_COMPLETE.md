## React Components

### Component Architecture

All React components are located in `client/src/components/samfox/`:

```
client/src/components/samfox/
â”œâ”€â”€ TemplateLoader.tsx      # Generic HTML template loader
â”œâ”€â”€ PaymentGateway.tsx      # Payment template wrapper with PayPal
â”œâ”€â”€ MarketplaceFrame.tsx    # Marketplace template renderer
â””â”€â”€ DashboardFrame.tsx      # Dashboard template renderer
```

### TemplateLoader.tsx

**Purpose:** Load any HTML template dynamically

**Props:**
- `category` - Template category (master, payment, marketplace, etc.)
- `template` - Template filename
- `className` - Optional CSS class
- `onLoad` - Callback when template loads
- `onError` - Callback on load error

**Example:**
```tsx
<TemplateLoader
  category="payment"
  template="global_checkout_form.html"
  onLoad={(html) => console.log('Loaded!')}
/>
```

### PaymentGateway.tsx

**Purpose:** Wrapper for payment templates with PayPal integration

**Props:**
- `template` - Payment template filename
- `amount` - Payment amount
- `currency` - Currency code (default: USD)
- `productName` - Product name
- `onSuccess` - Success callback
- `onError` - Error callback

**Example:**
```tsx
<PaymentGateway
  template="Fruitful_payment.html"
  amount={99.99}
  currency="USD"
  productName="Premium License"
  onSuccess={(data) => handleSuccess(data)}
/>
```

### MarketplaceFrame.tsx

**Purpose:** Render marketplace templates with product data

**Props:**
- `template` - Marketplace template filename
- `products` - Array of product objects
- `onProductClick` - Product click handler

**Example:**
```tsx
<MarketplaceFrame
  template="fruitful_innovations.html"
  products={productArray}
  onProductClick={(product) => viewDetails(product)}
/>
```

### DashboardFrame.tsx

**Purpose:** Display dashboard templates with real-time metrics

**Props:**
- `template` - Dashboard template filename
- `metrics` - Array of metric objects
- `refreshInterval` - Auto-refresh interval (ms)
- `onRefresh` - Refresh callback

**Example:**
```tsx
<DashboardFrame
  template="dashboardv3.0.html"
  metrics={metricsData}
  refreshInterval={30000}
  onRefresh={() => fetchNewData()}
/>
```

## PayPal API

### Configuration

**Environment Variables:**
```env
PAYPAL_CLIENT_ID=your_client_id_here
PAYPAL_CLIENT_SECRET=your_client_secret_here
PAYPAL_ENVIRONMENT=sandbox  # or 'production'
```

### Integration Methods

**1. Server-Side (Recommended)**

```typescript
import { createOrder, capturePayment } from '@/server/paypal/api';

// Create order
const order = await createOrder({
  amount: '99.99',
  currency: 'USD',
  intent: 'CAPTURE'
});

// Capture after approval
const payment = await capturePayment(order.id);
```

**2. Client-Side (Templates)**

```html
<script src="https://www.paypal.com/sdk/js?client-id=YOUR_CLIENT_ID"></script>
<script>
  paypal.Buttons({

    onApprove: function() { /* ... */ }

</script>
```

**3. React Components**

```tsx
import { PayPalButton } from '@/components/PayPalButton';

<PayPalButton
```

### Example 2: Payment Flow

```tsx
import { PaymentGateway } from '@/components/samfox/PaymentGateway';
import { useRouter } from 'wouter';

function CheckoutPage() {
  const [, navigate] = useRouter();

  const handleSuccess = (paymentData) => {
    console.log('Payment successful:', paymentData);
    navigate('/success');
  };

  return (
    <PaymentGateway
      template="global_checkout_form.html"
      amount={299.99}
      currency="USD"
      productName="Enterprise License"
      onSuccess={handleSuccess}
      onError={(err) => console.error('Payment failed:', err)}
    />
  );
}
```
